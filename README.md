# Surfn Papirus Casablanca icon theme

A variant of the [Surfn](https://github.com/erikdubois/surfn) icon theme. Ships only the
**Surfn-Papirus-Casablanca** theme; it inherits the base **Surfn** set, so `surfn-icons-git` is required.

## Installation

```
sudo pacman -S surfn-papirus-casablanca-icons-git
```

`surfn-icons-git` (base Surfn) is pulled in automatically as a dependency.

## Manual

Copy `usr/share/icons/Surfn-Papirus-Casablanca` into `~/.icons` (or `~/.local/share/icons` on Plasma),
ensure the base `Surfn` theme is present, then select **Surfn Papirus Casablanca** in your appearance settings.
